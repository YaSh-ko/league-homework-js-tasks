# Задачи на каррирование и преобразование строки в объект

## Задача 1 
Функция, которая позволяет себя последовательно вызывать для суммирования или иных операций, и возвращает резуьтат при выводе или математический операций. 

Реализовано с помощью каррирования функций и преобразования встренных методово. То есть, есть функция обертка, в которой есть начальный массив, куда будут класться все последующий аргументы функции. Есть сама каррированная функция, которая рекурсивно перебирает все последующие аргументы в функции. А с помощью переопределения встроенных методов Function, а именно valueOf() и toString() возрвращается результат при неявном вызове этих методов. Например, когда мы добавляем что-то (0 как в решении) или же превращаем ответ в строку. 

## Задача 2
Преобразование строки в объект, разделяя свойства по точке

Функция преобразует строку сначала в массив, затем дочерния функция рекурсивно проходит по каждому слову в массиве и возвращает из него объект, когда доходит до конца возвращает пустой объект.
